# A股智能选股系统设计方案（V1.0）

## 1. 项目背景

### 1.1 项目目标

构建一套基于实时新闻、上市公司公告、市场行情、产业链知识图谱以及AI分析能力的智能选股系统。

系统目标：

* 实时监控市场重大事件
* 自动识别热点题材
* 自动分析受益产业链
* 自动筛选真正受益公司
* 自动验证市场反馈
* 自动输出短线机会榜与长线价值榜

---

### 1.2 核心理念

传统选股逻辑：

```text
新闻
↓
推荐股票
```

本系统逻辑：

```text
新闻
↓
事件识别
↓
产业链分析
↓
受益程度评估
↓
市场验证
↓
股票评分
↓
排行榜输出
```

核心目标：

> 从大量利好消息中找到最值得投资的少数股票。

  "themes": [

  ],

  "industries": [

  ],

  "customers": [
    "腾讯",

  ]

## 4.2 股票画像字段

| 字段            | 描述   |
| ------------- | ---- |
| stock_code    | 股票代码 |

| industry      | 一级行业 |
| sub_industry  | 二级行业 |

| customers     | 下游企业 |
